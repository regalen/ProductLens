# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/16431670-1a25-4b69-bb5b-e7104746be62

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set `JWT_SECRET`
3. Run the app:
   `npm run dev`
